AuraLLM is a highly performant, commercial-grade AI Gateway written in Go. It serves as a secure, lightweight bridge connecting your developers' standard OpenAI codebases to any downstream LLM provider (OpenAI, Anthropic, Gemini, DeepSeek, LLaMA, etc.) while enforcing **real-time budget limits, local PII sanitization, and automated model migration audits** on-the-fly.

```python Python (From standard OpenAI Client to AuraLLM in 2 lines!)
import openai

# Simply swap URL and use your AuraLLM Gateway Key to get strict budget caps & PII scrubbing!
client = openai.OpenAI(
    api_key="gw-team-key-your-secret-here", 
    base_url="https://gateway.yourcompany.com/v1" 
)
```

## 🎯 Enforcing AI Governance by Stakeholder

Enterprise AI adoption requires consensus. AuraLLM satisfies all three core organizational pillars natively within the gateway proxy layer, eliminating the need for expensive multi-vendor security integrations:

<Tabs>
  <Tab title="CFO (FinOps & Budgets)">
    ### Enforce Spend Boundaries with Microsecond Precision
    
    AuraLLM gives financial leaders complete predictability over generative AI budgets:
    *   **Distributed Budget Caps:** Budgets are synchronized globally across your clusters using a distributed **Redis Cache** with atomic float increments.
    *   **Proactive Webhook Alerts:** Receive instant notifications via **Slack / PagerDuty** the second a team consumes 80% or 100% of their allocated budget.
    *   **Anti-Spam Threshold Locks:** Restricts budget warnings to exactly once per threshold per team per month, preserving your Slack channels.
  </Tab>
  <Tab title="CISO (Data Security & Compliance)">
    ### Local PII Sanitization & SOC2/HIPAA Readiness
    
    Stop employees and apps from leaking customer data or company secrets to public LLM servers:
    *   **Local Compiled Regex Scrubber:** Automatically detects and scrubs Social Security Numbers (SSNs), Credit Cards, emails, and proprietary API Keys locally on your private servers.
    *   **Auto Re-injection (Un-redaction):** Original sensitive values are securely held in local gateway memory and re-injected back into standard model replies on-the-fly.
    *   **Frictionless Compliance:** Satisfy SOC2, HIPAA, and GDPR audit trails without developers writing a single line of custom security code.
  </Tab>
  <Tab title="CTO (DevOps & Engineering)">
    ### Raw Go Concurrency, High Availability, and No Vendor Lock-In
    
    Give your developers complete model freedom without sacrificing infrastructure speed:
    *   **Shadow Routing (Migration Auditor):** Duplicate live production prompts in background goroutines to test cheaper models side-by-side with zero reliability risk or added latency.
    *   **Zero-Dependency Binary:** AuraLLM compiles down to a single ~20MB executable running on only **~15MB of RAM** with startup latency under **10ms**.
    *   **OIDC/Okta SSO Integration:** Gated under a secure admin portal with Role-Based Access Controls to enforce separation of duties.
  </Tab>
</Tabs>
